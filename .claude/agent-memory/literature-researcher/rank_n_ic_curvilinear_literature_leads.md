When asked "has anyone derived rank-N interface-current closures for
curvilinear (cylinder/sphere) cells," these three papers are the only
title-level matches returned by CrossRef-filtered searches of Annals
of Nuclear Energy and NSE:

1. **Bogado Leite, S.Q. (1998)** — "Revised interface-current relations
   for the unit-cell transport problem in cylindrical and spherical
   geometries," ANE 25(6). DOI 10.1016/S0306-4549(97)00026-1.
   Orphaned: 1 citation total, 0 influential (Semantic Scholar,
   verified 2026-04). No abstract exposed by OpenAlex/CrossRef/S2.
   Must be read from the PDF to assess content — the title is
   domain-exact but downstream impact is zero, which is unusual and
   warrants caution before citing.

2. **Krishnani, P.D. (1982)** — "Interface current method for PHWR
   cluster geometry with anisotropy in the angular flux at interfaces,"
   ANE 9(5). DOI 10.1016/0306-4549(82)90083-4. Explicit anisotropic
   IC, cluster (not hollow shell) geometry.

3. **Mohanakrishnan, P. (1982)** — "Choice of angular current
   approximations for solving neutron transport problems in 2-D by
   interface current approach," ANE 9(5). DOI
   10.1016/0306-4549(82)90030-5. Closest title match for
   "basis-scale calibration / gauge DOF" concept; 2D Cartesian.

**Adjacent for 1/ρ Peierls derivations**:
- Corngold 2002 + 2004 addendum (ANE) — cylinder CP Peierls algebra.
- Wio 1984 + Krishnani 1985 — CP kernel transformation laws under
  geometric rescaling.

**Four-reference synthesis covers the rest**: Ligou 1982 Ch.8 +
Sanchez 2002 + Stamm'ler 1983 Ch.IV + Stacey 2007 Ch.9 all use
scalar/DP-0 for curvilinear cells. No Eddington-weighted adaptive
closure in the standard textbook corpus. See
rank_n_closure_four_references_synthesis.md.

**How to apply**: If the user asks "has anyone done X for rank-N
curvilinear IC," these three ANE papers are the last stop before
concluding "probably novel." Bogado Leite 1998 is the most
dangerous false-negative — its zero-citation status and hidden
abstract mean we can't rule it out from metadata alone.
